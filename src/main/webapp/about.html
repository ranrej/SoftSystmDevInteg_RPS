About Rock-Paper-Scissors Game



Overview

The Rock-Paper-Scissors Game is a full-stack web application that allows users to play the classic game of Rock-Paper-Scissors in real-time using WebSockets. Players can connect to the game, choose their move, and see the result instantly. The application is built using HTML, CSS, JavaScript, and WebSocket.


Features

  • Real-time gameplay between two players
  • Connect and play instantly
  • Choose between rock, paper, or scissors
  • See the result of each round immediately
  • See the scoreboard update immediately
  • Option to restart the game

How to Play

To play the Rock-Paper-Scissors Game, follow these steps:

  1. 1. Clone the repository: `git clone https://github.com/OntarioTech-CS-program/w24-csci2020u-final-project-mohsin-rejdal-badwal-oudahmane.git`
  2. 2. Open your web browser and go to `http://localhost:3000`
  3. 3. Select your weapon and wait for the other player to select theirs.
  4. 4. You need 5 points to win the match.
  5. 5. Select 'Play again' to start a new match.
